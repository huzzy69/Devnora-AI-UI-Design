## ğŸ› ï¸ **DEPLOYMENT GUIDE**

### Step 1: Export Your Project
1. Go to `/dashboard/deployments`
2. Click "Export Project"
3. Download ZIP file
4. Extract on your computer

### Step 2: Prepare Hosting
1. Choose PHP shared hosting provider:
   - Hostinger (Recommended)
   - Bluehost
   - NameCheap
   - SiteGround
   - Any PHP 7.4+ hosting

2. Requirements:
   - PHP 7.4 or higher
   - MySQL 5.7 or MariaDB 10.3+
   - 100MB+ storage
   - SSL certificate (free with Let's Encrypt)

### Step 3: Upload Files
1. Connect via FTP/SFTP:
   - Use FileZilla or similar
   - Connect to your hosting
   - Navigate to public_html/

2. Upload ZIP contents:
   - Upload all files from extracted ZIP
   - Maintain folder structure
   - Set correct permissions (755 for folders, 644 for files)

### Step 4: Configure Database
1. Create MySQL database via cPanel
2. Import database.sql file
3. Update config.php with credentials:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'your_database');
   define('DB_USER', 'your_username');
   define('DB_PASS', 'your_password');
   ```

### Step 5: Configure Application
1. Update .env file:
   ```
   APP_URL=https://yourdomain.com
   APP_ENV=production
   APP_DEBUG=false
   ```

2. Set up cron jobs (for workflows):
   ```
   */5 * * * * php /path/to/cron.php
   ```

### Step 6: Test Deployment
1. Visit your domain
2. Complete installation wizard
3. Create admin account
4. Test core features

### Step 7: SSL & Security
1. Install SSL certificate (free Let's Encrypt)
2. Force HTTPS in .htaccess
3. Set secure file permissions
4. Change default admin password
